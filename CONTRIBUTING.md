# Hur du bidrar till DevTips

Tack för att du vill bidra! Följ dessa steg:

1. Forka detta repo
2. Skapa en branch: `feature/ditt-tips-namn`
3. Lägg till ditt tips som ett `<li>`-element i `index.html`
4. Commita med referens till issue: `feat: lägg till [ämne]-tips, closes #X`
5. Pusha till din fork
6. Skapa en Pull Request

## Regler
- Ett tips per PR
- Följ HTML-formatet som redan finns
- Skriv tydliga commit-meddelanden
- Var artig i kommentarer
